## 🎯 Visão Geral

Este portfólio foi desenvolvido por **Guilherme Oliveira**, um desenvolvedor backend focado em construir **APIs escaláveis, seguras e de alto desempenho** utilizando Python, JavaScript e SQL.

O design é moderno, limpo e profissional, com foco em:
- ✅ Experiência do usuário (UX) intuitiva
- ✅ Design responsivo (mobile, tablet, desktop)
- ✅ Acessibilidade (WCAG compliant)
- ✅ Performance otimizada
- ✅ Suporte multilíngue

---

## ✨ Características

### 🎨 Design e UX
- **Paleta de Cores Moderna**: Gradiente laranja/vermelho profissional
- **Modo Escuro/Claro**: Toggle com persistência em localStorage
- **Layout Sidebar**: Navegação intuitiva e sempre visível
- **Animações Suaves**: Transições elegantes entre seções
- **Design Responsivo**: Adapta-se perfeitamente a qualquer tela

### 🌍 Multilíngue
- **Suporte PT-BR e EN**: Alternância rápida entre idiomas
- **Persistência**: Idioma salvo em localStorage
- **Cobertura Completa**: Todas as seções traduzidas

### 🔒 Segurança e Performance
- **Sem Dependências Externas**: Apenas Bootstrap Icons (CDN)
- **Código Limpo**: JavaScript vanilla, sem frameworks
- **SEO Otimizado**: Meta tags e HTML semântico
- **Validação de Links**: Rel attributes para segurança

### 💾 Persistência de Dados
- **Tema Salvo**: Preferência dark/light mantida entre sessões
- **Idioma Salvo**: Último idioma selecionado restaurado automaticamente
- **localStorage API**: Sem necessidade de backend

## 🛠️ Tecnologias

### Core
- **HTML5** - Markup semântico e acessível
- **CSS3** - Custom properties, Grid, Flexbox, Media Queries
- **JavaScript (ES6+)** - Classes, Arrow Functions, Async/Await

### Bibliotecas
- **Bootstrap Icons 1.11.0** - Ícones profissionais via CDN
- **Nenhuma dependência npm**: Tudo rodando vanilla

### Ferramentas de Desenvolvimento
- **VS Code** - Editor recomendado
- **Git/GitHub** - Controle de versão
